Errata - Aula 2 - Computação II (DCC - UFRJ)

O compilador java (testado na versão 12.0.1) não faz verificação estática de limite de arrays. Portanto, o código que foi mostrado em aula:

public class LimiteArray {
    public static void main(String[] args) {
        int[] v = new int[100];
        v[100] = 2;
    }
}

Compilará sem erros, mas causará uma exceção (java.lang.ArrayIndexOutOfBoundsException) quando for executado.
