Exercícios 2 - Aula 3 - Comp II (DCC - UFRJ)

Atenção: Escrever Javadoc para todos os programas.

  1. Analise o trecho de código abaixo.

     float perc = 0.0f;
            
 for (int i = 0; i < 100; i++) {
     // Faz operação demorada aqui usando o índice...
        
     perc += 0.01;
     System.out.printf("%d completado \n", (int)(perc * 100));
 }

    Responda:

    a) Qual é o objetivo do código que podemos inferir a partir de sua análise?

    b) O código realiza o objetivo corretamente? Porquê?


  2. Implemente um programa que receba da entrada padrão dois vetores de números reais (um vetor port linha, em Rn), calcule o ângulo entre eles e escreva o ângulo, em graus, na saída padrão.
    (Dica: Usar o método nextDouble(), da classe Scanner)

  3. Consulte a documentação da linguagem java e outros sites, como o tutorial oficial da linguagem e responda:

    a) Qual classe representa a exceção disparada quando:

     I. Tentamos alocar mais memória que o disponível.

 II. Tentamos instanciar um array passando como tamanho uma variável que contém um valor negativo.

    b) Que classes podemos utilizar para representar o tempo (data, hora)? Como podemos utilizá-las?

    c) Como podemos implementar uma pilha de números em um programa Java? Escreva um exemplo.
    (Dica: A pilha é uma estrutura onde ultimo elemento a ser colocado nela é o primeiro a ser acessado)
